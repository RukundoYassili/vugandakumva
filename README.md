# Vugandakumva — Speak, I Can Hear

A GBV (Gender-Based Violence) awareness initiative founded by David Niyomugabo in 2025, dedicated to education, prevention, and support.

## Pages

- **Home** — Mission overview and key statistics
- **About** — Story and vision behind the initiative
- **Awareness** — GBV education and prevention resources
- **Resources** — Support services and helplines
- **Contact** — Get in touch

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js + Express
- **AI Chatbot:** Google Generative AI / OpenAI
- **Database/Auth:** Supabase
- **Hosting:** Render

## Getting Started

```bash
# Install dependencies
npm install

# Create a .env file with your API keys
# (Supabase, Google AI / OpenAI)

# Start the server
npm start
```

The app will run at `http://localhost:3000` by default.
